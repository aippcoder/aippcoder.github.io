为GitPage绑定自己的域名

发表于 2017-03-11 | 分类于 杂记 |

使用GitPage的方式搭建博客域名通常为username.github.io，有时候我们想使用自己的域名作为自己博客的站点，这时候就需要为GitPage绑定域名了，毕竟使用自己域名的网站逼格看起来要高一些，而且普通的域名也比较廉价。域名购买不做说明，如果需要购买可以在万网或者一些其他的域名提供商那里购买，购买之后需要的就是解析了。这里使用的是阿里云作为DNS提供商。

绑定域名可以分为四步走：

  1. 在项目的根目录下创建一个CNAME文件，内容为需要绑定的域名，比如yoursite.com

  2. ping你的 http://username.github.io ，得到一个IP；比如ping username.github.io

  3. 修改你的域名解析记录，用得到的ip添加一个A记录，并同时添加另外两个github的ip(192.30.252.153,192.30.252.154)记录，

  4. 登录http://github.com 进入项目>Settings>Custom domain>输入你的域名>Save。

这样配置就完成了，测试是否绑定成功。
